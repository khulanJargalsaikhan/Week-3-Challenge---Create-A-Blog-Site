Introduction to Java

23 July 2021

Lecture Notes

Java is a programming language developed at Sun Microsystems in 1995 by James Gosling.

There are several editions of Java; the latest is version 8 (referred to as 1.8). These editions are categorized into Standard (SE), Enterprise (EE), and Mobile (ME).

Java is considered a write once, run anywherelanguage, because the same code can be executed on different operating systems.

Common features of Java
  • object-oriented
  • platform independent
  • simple; easy to learn
  • secure
  • architectural-neutral
  • portable
  • robust
  • multi-threaded
  • interpreted
  • high performance
  • distributed
  • dynamic
Short History of Java
  • 1991: initiated by James Gosling
  • 1995: initial release
  • 1998: Java 2 (J2SE) released
  • 2006: Java released as open-source
    • J2SE, J2EE, J2ME and others renamed as JSE, JEE and JME
  • 2009-2010: Oracle acquires Sun (and Java)
